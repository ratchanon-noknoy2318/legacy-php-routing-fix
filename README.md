# Kosamphi ITA 2024: Building Trust through Transparency

This project was created during my internship at the Office of the District Public Health, Kosamphinakhorn. It is a dedicated system built to ensure honesty, transparency, and easy access to information for the community.

---

### 1. The Heart of the Project (Purpose)
I designed this system to help the public health office maintain high standards of integrity and make government documents easy for everyone to see.

| Current Goal | How I helped | Warm & Clear Result |
| :--- | :--- | :--- |
| **Be Transparent** | Public Disclosure Hub | Everyone can see important news easily |
| **Keep Data Safe** | Secure Database Design | Personal records are protected with care |
| **Follow Rules** | Assessment Tracking | The office meets all official standards |
| **Easy Access** | Document Management | No more lost papers; everything is online |

---

### 2. Built with Care (Simple Design)
The system is made to be reliable and friendly for both staff and the public.

| Part | Simple Explanation | Main Benefit |
| :--- | :--- | :--- |
| **Website Face** | Clean HTML & CSS | Easy to read and navigate for everyone |
| **Information Center** | MySQL Database | Stores and organizes data safely |
| **Security** | Safe Login System | Protects information from unauthorized access |
| **Server** | MOPH Official Hosting | Runs on a trusted hospital-grade system |
